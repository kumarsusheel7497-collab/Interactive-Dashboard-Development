

# Interactive Dashboard Development

## Overview
This project focuses on building an interactive dashboard to present key insights clearly and effectively. The dashboard uses visual elements such as bar charts, line charts, pie charts, and summary metrics to communicate trends, performance indicators, and actionable insights for better decision-making.

## Objective
The main objective of this project is to create an interactive dashboard that transforms raw or cleaned data into meaningful visual insights. It helps users understand performance trends, important metrics, and business patterns in an easy-to-read format.

## Tools Used
- Python
- Pandas
- NumPy
- Plotly
- Google Colab

## Steps Performed
- Loaded the dataset into Google Colab
- Inspected rows, columns, missing values, and duplicates
- Removed duplicate rows
- Cleaned and standardized column names
- Handled missing values
- Calculated key summary metrics
- Created interactive bar charts for category-wise analysis
- Built line charts for trend analysis
- Used pie charts for distribution analysis
- Generated histograms for numeric distributions
- Created an interactive dashboard HTML file
- Saved the cleaned dataset for future use

- `interactive_dashboard.html` → Interactive dashboard file

## Key Insights
The dashboard helps visualize important metrics and patterns in the dataset. It provides a quick understanding of trends, category performance, and overall business behavior through interactive charts and summary indicators.
